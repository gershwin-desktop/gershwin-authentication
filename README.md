# Gershwin Authentication
This provides gsauth that presents an authentication window for Gershwin Applications

Prerequisites

```
sudo pkg install -g 'GhostBSD*-dev'
sudo pkg install gershwin-developer
```

## Build Instructions

### Build
```bash
gmake
```

### Install
```bash
sudo gmake install 
```

### Run
```bash
gsauth --exec pkg update
```
